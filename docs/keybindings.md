# Keybindings Reference

Mod key is Super (Win key).

## awesome WM — System

| Binding | Action |
|---------|--------|
| `Mod+Ctrl+r` | Reload awesome config |
| `Mod+Shift+q` | Quit awesome (back to display manager) |
| `Mod+Ctrl+l` | Lock screen (slock) |
| `Mod+s` | Show all hotkeys popup |

## awesome WM — Launchers

| Binding | Action |
|---------|--------|
| `Mod+Return` | Open terminal (alacritty) |
| `Mod+d` | Open app launcher (rofi) |
| `Mod+e` | Open file manager (ranger in alacritty) |
| `Mod+r` | Run prompt in status bar |
| `Print` | Screenshot (full screen) → ~/screenshots/ |
| `Shift+Print` | Screenshot (select region, flameshot) |

## awesome WM — Workspaces

| Binding | Action |
|---------|--------|
| `Mod+1` | Switch to workspace `OSI` |
| `Mod+2` | Switch to workspace `term` |
| `Mod+3` | Switch to workspace `web` |
| `Mod+4` | Switch to workspace `tools` |
| `Mod+5` | Switch to workspace `recon` |
| `Mod+6` | Switch to workspace `exploit` |
| `Mod+7` | Switch to workspace `post` |
| `Mod+8` | Switch to workspace `files` |
| `Mod+9` | Switch to workspace `misc` |
| `Mod+Left` | Previous workspace |
| `Mod+Right` | Next workspace |
| `Mod+Escape` | Last visited workspace |
| `Mod+Shift+1-9` | Move focused window to workspace N |
| `Mod+Ctrl+1-9` | Toggle workspace N on current screen |

## awesome WM — Window Focus

| Binding | Action |
|---------|--------|
| `Mod+j` | Focus next window |
| `Mod+k` | Focus previous window |
| `Mod+Tab` | Focus last focused window |
| Click | Focus window under cursor |

## awesome WM — Window Management

| Binding | Action |
|---------|--------|
| `Mod+Shift+j` | Swap with next window |
| `Mod+Shift+k` | Swap with previous window |
| `Mod+f` | Toggle fullscreen |
| `Mod+m` | Toggle maximize |
| `Mod+n` | Minimize window |
| `Mod+t` | Toggle always-on-top |
| `Mod+Shift+c` | Close window |
| `Mod+Ctrl+Space` | Toggle floating |
| `Mod+1` (click drag) | Move floating window |
| `Mod+3` (click drag) | Resize floating window |

## awesome WM — Layout

| Binding | Action |
|---------|--------|
| `Mod+Space` | Next layout |
| `Mod+Shift+Space` | Previous layout |
| `Mod+l` | Grow master pane |
| `Mod+h` | Shrink master pane |

Available layouts in order: tile, tile-left, tile-bottom, fair, spiral-dwindle, max, floating.

## tmux

Prefix is `Ctrl+a`.

| Binding | Action |
|---------|--------|
| `Prefix+|` | Split pane vertically |
| `Prefix+-` | Split pane horizontally |
| `Prefix+h/j/k/l` | Navigate panes (vim-style) |
| `Prefix+H/J/K/L` | Resize pane |
| `Prefix+r` | Reload tmux config |
| `Prefix+[` | Enter copy mode (vi keys) |
| `v` (copy mode) | Begin selection |
| `y` (copy mode) | Copy selection |
| `Prefix+]` | Paste |
| `Prefix+c` | New window |
| `Prefix+n/p` | Next/previous window |
| `Prefix+1-9` | Jump to window N |
| `Prefix+,` | Rename window |
| `Prefix+$` | Rename session |
| `Prefix+d` | Detach session |

## vim

Leader key is `\` (backslash).

| Binding | Action |
|---------|--------|
| `Enter` | Clear search highlight |
| `\w` | Save file |
| `Ctrl+w hjkl` | Navigate splits |
| `Ctrl+w v` | Vertical split |
| `Ctrl+w s` | Horizontal split |
